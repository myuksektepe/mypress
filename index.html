# WAMP Server Nedir?

Bildiğiniz üzere PHP sunucu  tabanlı çalışan bir programlama dilidir. Sunucu içerisinde PHP çalıştırabilmek için uygun ortamın hazırlanmış olması gerekmektedir. Bu ortamı sağlayacak hazır paket programlar bulunmaktadır. Bu makalemizde bu programlar içerisinde en çok tercih edilen WAMP Server’dan, kurulumu ve kullanımından bahsedeceğiz.

WAMP Server (Windows, Apache, MySQL, PHP)

 

Windows işletim sistemine sahip sunucularda kullanılmak üzere Linux web sunucularında kullanılan LAMP paketinden türetilerek oluşturulmuştur.

 

Başarılı bir kurum sonrası bizlere;

  • Apache,
  • PHP,
  • MySQL (veri tabanı sunucusu) ve
  • PhpMyAdmin (veri tabanı yönetim ara yüzü)

 

Sistemlerini sağlar ve bunları tek yerden yönetmemize olanak tanır. Kendi web sitesi (http://www.wampserver.com/en/) üzerinden ücretsiz olarak dağıtılmaktadır.

# WAMP Server ile Neler Yapabiliriz?

Elinizde PHP dili kodlanmış bir web siteniz ve Windows işletim sistemine sahip bir sunucunuz varsa web sitenizi en iyi şekilde yayınlamak için WAMP Server biçilmiş kaftan.

 

Ayrıca MySQL sayesinde ile veri tabanlarınızı oluşturup yönetebilirsiniz. Böylece veri saklayabilen web siteleriniz, -yazacağınız API servisleri sayesinde- mobil uygulamalarınız ya masaüstü programlarınız olabilir.

 

Bu veri tabanlarını oluşturmak ve kullanmak için de sizlere sağlanan PhpMyAdmin ara yüzü sayesinde işleriniz olabildiğince kolaylaşmış olacaktır.

Kurulum aşamasında klasik kurulum ekranlarını “İleri” butonu ile geçiyoruz.

 

 

 

 

 

Kurulumun yapılacağı dizini seçiyoruz. Genelde burada yazılı olan dizini değiştirmenize gerek kalmaz. Ancak bazı durumlarda yedek diskiniz olan E sürücüsünü seçmek isteyebilirsiniz.

 

 

 

 

 

 

Kurulumun sonuna doğru karşınıza birkaç soru penceresi çıkacaktır. Bu ilk pencerede size Internet Explorer’dan başka kullanmak istediğiniz web tarayıcısı olup olmadığını soruyor. Google Chrome yada Mozilla Firefox gibi bir tarayıcı birincil olarak seçmek için “Evet” butonuna tıklayın ve dosya yolunu belirtin.

 

 

 

 

Sonraki soru penceresinde ise kodlama yaparken kullanacağınız editör programını Windows’un kendi Not Defteri uygulaması dışında başka bir program olarak seçip seçmeyeceğinizi soruyor. Sublime Text, Atom, Visual Studio Code vb. gibi bir IDE seçmek için bu seçeneğe de “Evet” deyip dosya yolunu belirtiyoruz.

 

 

Birkaç kere daha İleri seçeneği tıkladıktan sonra “Bitir“ diyoruz ve WAMP Server başarıyla kurulmuş oluyor.

 

 

Çalışıp çalışmadığını test etmek için web tarayıcımızın adres çubuğuna http://localhost yazabiliriz. Her şey yolunda gittiyse karşımıza açılacak olan sayfanın alttaki gibi bir görüntüye sahip olması gerekiyor.

 

 

# WAMP Server'ı Tanıyalım

WAMP Server kurulum esnasında seçtiğiniz dizine www isimli bir klasör oluşturur. Bu dizine gittiğinizde tarayıcınıza http://localhost yazdığınızda görüntülenen sayfanın dosyalarına ulaşmış olursunuz. Bu da; bundan sonra yapacağımız web sitelerini bu klasörün altında toplamamız gerekmekte demektir. Hemen bir deneme.php dosyası oluşturun ve web tarayıcınıza http://localhost/deneme.php yazıp sonucu görüntüleyin.

 

 

 

 

WAMP Server’ı başlattıktan sonra simgesi sağ aşağı tarafta görünmeye başlayacaktır. Servislerin hazırlanma sürecinde bu simge kırmızı görüntülenir. Servisler doğru şekilde hazırlanır ve kullanıma hazır hale gelirse yeşil hale gelecektir.

 

 

 

Simgeye sağ tıkladığımızda WAMP Server ayarlarını yapabildiğimiz pencere açılacaktır. Bu penceredeki seçeneklerden dil ayarı, WAMP Server’ı kapatma ve yenileme gibi işlemleri tek tıklama ile yapabileceğiniz gibi,

 

 

Wamp Settings (Ayarlar) menüsünden; Wamp Server’ın çalıştığı lokal IP ayarları, veri tabanı servislerinin seçimleri, lokal de çalışacak olan hosting servisinin link yapısı gibi ayarları yapabilirsiniz.

 

 

 

Ayrıca Tools (Araçlar) menüsünden;

Apache’nin, MySql’in ve MariaDB’nin dinledikleri portları değiştirebilirsiniz.

Dikkat: Eğer bilgisayarınızda Skype yüklü ise Skype’ın kullandığı port ile Apache’nin kullandığı 80 portu çakışabilir. Bu durumda ikisinden birinde değişiklik yapmak zorunda kalabilirsiniz. Ama böyle bir durum söz konusu değilse bu ayarları –gerekmedikçe- değiştirmeyin.

 

 

 

 

WAMP Server simgesine sol tuşla tıkladığımız zaman ise karşımıza servislerin detaylı ayarlarına ulaşabildiğimiz aynı zamanda çalışan servisleri durdurup, yeniden başlatabildiğimiz bir seçenekler penceresi çıkacaktır.

 

 

Buradan her servis için ayrı ayrı ayarlamalar yapabiliriz.

 

 

Örneğin; www klasörü içine taşıdığımız web sitemizi görüntülemeye çalışırken hatalar alabiliriz. Bu hataların WAMP Server’da seçili olan PHP sürümü ile web sitemizin PHP sürümünün uyuşmazlığından kaynaklandığını varsayalım. Bu hatayı düzeltmek için yapmamız gereken şey çok basittir, WAMP simgesine sol tıklayıp PHP > Version menüsünü takip etmek ve uygun olan PHP sürümünü seçmek.  (Alttaki görseli inceleyin)

# Kurarken Karşılaşılabilecek Hatalar ve Çözümleri

 

php-win.exe – MSVCR110.dll Hatası ve Çözümü

Hata Adı:

EN: php-win.exe System Error

TR: php-win.exe Sistem Hatası

 

Açıklama:

EN: The program can’t start because MSVCR110.dll is missing from your computer.

TR: Bu program çalıştırılamaz çünkü bilgisayarınızda MSCVR110.dll dosyası bulunamadı.

 

Çözüm:

Windows işletim sistemli sunucu veya bilgisayarlarda bazı programları ve sistemleri çalıştırabilmek için Windows’un şart koştuğu kütüphane dosyalarının yüklü ve güncel olması gerekmektedir.

Bu işlem için aşağıdaki adımları takip ederek gerekli dosyaya ulaşın ve yükleme işlemini yapın.

 

Adım 1 – Wamp Server’ı Program Ekle/Kaldır’dan kaldırın.

Adım 2 – Aşağıda belirttiğimiz dosyayı indirin, yükleyin ve bilgisayarınızı yeniden başlatın.

Çalışan Servislerin Eksik Kalması yada Bütün Servislerin Çalışmaması (Turuncu Simge)

 

 

Açıklama:

Eğer php-win.exe- MSVCR110.dll, httpd.exe - VCRUNTIME140.dll, mysqld.exe – MSVCR110.dll  hatalarını aldıysanız ve anlatıldığı gibi çözüme ulaştıysanız büyük ihtimalle WAMP Server ikonunu turuncu şekilde görüyorsunuz. Bunun sebebi çalışması gereken servislerde eksiklik olmasıdır. PHP, Apache ve MySql servislerinin biri bile çalışmıyor olursa bu turuncu ikonu görmeye devam edersiniz.

 

Servislerin çalışmama sebebi ise yükleme esnasında karşılaşılan hatalar sonucu doğru yüklenememesi yada hiç yüklenememesidir. Bu durumda Wamp Server Manager’dan ayarlarına gidip ilgili servisi yeniden yüklememiz gerekebilir.

 

Aşağıdaki adımları takip ederek bu sorunları çözebilirsiniz.

 

  • Apache Servis Yükleme

 

Wamp Server simgesine sol tıklayın ve (görseldeki gibi) Apache > Services > Install Service adımlarını takip edin.

 

 

 

 

  • MySql Servis Yükleme

 

Wamp Server simgesine sol tıklayın ve (görseldeki gibi) MySql > Services > Install Service adımlarını takip edin.

# İnternete Açmak (Online Hale Getirmek)

Aşağıdaki adımları takip ederek sunucunuza yada bilgisayarınıza yüklediğiniz WAMP Server’ı internete açık hale getirebilir yani lokalde barındırdığınız web siteleri online’a açabilirsiniz.

 

Adım 1: Put Online’ı Menüye Eklemek

WAMP Server Yöneticisine  “put online” ikonunu getirmeniz gerekmektedir. Bunun için görseldeki adımları takip ediniz yada Wamp Server ikonuna sağ tıklayın -> Wamp Settings -> Menu item: Online / Offline seçeneğine tıklayın.

Bu işlemi yaptıktan sonra Wamp Server ikonununa sol tıkladığınızda en altta “Put Online” seçeneğinin gelmiş olduğunu göreceksiniz.

 

 

 

 

 

Adım 2: Put Online İşlemini Başlatmak

Şimdi menüye yeni gelmiş olan “Put Online” seçeneğine tıklayın. Konsole ekranı açılıp kapandıktan sonra Wamp Server servisleri yeniden başlatacaktır. Hiçbir sorunla karşılaşmazsanız Wamp Server Online servisi doğru çalışırsa ikonu değişecek ve görseldeki gibi kırmızı nokta içeren bir hal alacaktır.

 

 

 

 

Adım 3: Kontrol Edelim

Bu işlemleri yaptıktan sonra başka herhangi bir bilgisayarda web tarayıcınıza sunucunuzun yada bilgisayarınızın IP Adresini girin. (IP Adresinizi öğrenmek için tıklayın) Görseldeki gibi bir ekranla karşılaşıyorsanız doğru yolda ilerliyorsunuz demektir.

 

Karşılaştığınız bu hata sunucuzun dinlediği 80 portunun dışarıdan gelen bağlantılara kapalı olduğu anlamına gelmektedir. Bunun birden fazla sebebi olabilir. En genel sebep sunucunuzdaki yada bilgisayarınızda ki Firewall ve Antivirüs yazılımlarının gelen istekleri engelliyor olmasıdır.

 

 

 

 

 

 

Adım 4: Windows Firewall’u Ayarlama:

Kontrol Paneline gidin - > Windows Firewall menüsünü bulun ve tıklayın -> Sol Menüden Windows Firewall Kapat / Aç ‘ı tıklayın ->Açılacak ekrandan Public network location settings ayarını kapalı hale getirin.

 

 

 

 

 

Adım 5: httpd-vhost.conf dosyasını düzenleme

 

Eğer buraya kadar adımları doğru şekilde uyguladıysanız Wamp Server’ı online hale getirmek için yapmamız gereken son bir adım kalmış oluyor. Bu adımda Apache servisinin dinlediği 80 Portunun dışardan gelen bağlantılara açılma işlemini gerçekleştireceğiz.

 

Bunun için şu yolu takip ederek gerekli dosyaya ulaşmamız gerekiyor;

wamp64\bin\apache\apache{version}\conf\extra\httpd-vhosts.conf

 

Dosyayı bulduktan sonra herhangi bir editör ile düzenlemek için açın. Karşınıza gelen kodlar aşağıdaki gibi olacaktır;

 

# Virtual Hosts

			#

			<VirtualHost *:80>

			  ServerName localhost

			  ServerAlias localhost

			  DocumentRoot "${INSTALL_DIR}/www"

			  <Directory "${INSTALL_DIR}/www/">

			    Options +Indexes +Includes +FollowSymLinks +MultiViews

			    AllowOverride All

			    Require local

			  </Directory>

			</VirtualHost>

 

Değiştirmemiz gereken satır kırmızı ve kalın harfler ile işaretli satırdır. Burada ki “local” değerini “all granted” ile değiştirip dosyayı kaydediyoruz.

 

# Virtual Hosts

			#

			<VirtualHost *:80>

			  ServerName localhost

			  ServerAlias localhost

			  DocumentRoot "${INSTALL_DIR}/www"

			  <Directory "${INSTALL_DIR}/www/">

			    Options +Indexes +Includes +FollowSymLinks +MultiViews

			    AllowOverride All

			    Require all granted  <-----

			  </Directory>

			</VirtualHost>

 

 

 

 

 

 

 

Adım 6: Wamp Server’ı Yeniden Başlatın

Wamp Server ikonuna sol tıklayın ve Restart All Services seçeneğine tıklayın.

 

 

 

 

 

 

Adım 7: Kontrol Edelim

Wamp Server ikonu tekrar kırmızı nokta içeren hale geldikten sonra IP Adresinizi başka bir bilgisayarda web tarayıcıya girin ve sonucu kontrol edin.

Localhost’de ki index.php içeriğini görüntüleyebiliyorsanız. İşlem başarıyla tamamlandı demektir.
